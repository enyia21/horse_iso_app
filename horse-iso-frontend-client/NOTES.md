## FRONTEND ##

### CREATE REACT APP###
* TO CREATE APP START WITH:

1. npm install -g npm@latest
2. nvm install node
3. npm install -g yarn

4.  run npx create-react-app <your app name>

### ADD REACT AND REDUX DEPENDIENCES ###

* ADD THE FOLLOWING:
    1. redux
    2. react-redux
    3. redux-thunk
    4. react-router-dom
* USE THE COMMAND:
    yarn add redux react-redux redux-thunk react-router-dom
